About E-mesg 💬

📝 Description

E-mesg is a next-gen messaging platform inspired by WhatsApp — built with a modern, clean UI and designed for future AI-powered features. Whether you're chatting, updating your status, or planning a voice/video call, E-mesg is your stylish, smart companion for seamless communication.

🚧 Prototype Features (Developed & Functional)

  • 🌙 Light/Dark theme toggle
  • 📱 Mobile-first responsive UI design
  • 🔁 Easy navigation between Chat, Status, and About pages
  • ⭐ Smart status updates with sorting
  • 📊 Chat sorting & filtering by date and group
  • ⚙️ Backend-ready structure prepared for Firebase or Node.js integration

🚀 Original Model Features (Planned & Upcoming)

  • 💬 Real-time chat interface with message delivery and read receipts
  • 📞 Voice & video calling integration
  • 📂 File and image sharing support
  • 🧠 AI tone-control and smart reply suggestions
  • 🔐 End-to-end encryption for privacy and security
  • 📍 Location sharing & map integration
  • 🤖 AI assistant/chatbot for enhanced conversation support

📘 User Manual

  • Accessing E-mesg: Open the landing page (index.html) to see the main navigation options: Chat, Status, and About.
  • Theme Switching: Use the 🌙/☀️ toggle button found in the top-right corner of the interface to switch between light and dark modes for comfortable viewing.
  • Navigation: Click on the navigation links or buttons to move between the Chat, Status, and About pages smoothly.
  • Status Updates: In the Status page, you can view and sort smart status updates by date or relevance.
  • Chat Sorting: On the Chat page (prototype), you can sort and filter chats by date or group for easy access.
  • Future Features: Real-time messaging, voice/video calls, file sharing, and AI-powered smart replies will be added in upcoming updates.
  • Privacy & Security: The platform is designed with user privacy in mind and will implement end-to-end encryption in the future.
  • Contact & Support: For any issues or feedback, reach out to the developer via the email below.
